# Web Automation

In this project, I have configured two leading frameworks for web test automation - [Cypress](https://www.cypress.io/) and [Playwright](https://playwright.dev/), using [TypeScript](https://www.typescriptlang.org/). The project includes several GUI tests covering the handling of various web elements and test scenarios, allowing for a comparison of the solutions provided by each framework. Additionally, the project features a few simple REST API tests aimed at comparing backend testing approaches.
