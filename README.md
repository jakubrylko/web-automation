# Web Automation

In the web automation project, I have set up two leading frameworks for web test automation - Cypress and Playwright, using TypeScript. The project includes several GUI tests covering the handling of various web elements and different test scenarios, allowing for a comparison of the solutions provided by each framework. Additionally, the project also features simple REST API tests.
